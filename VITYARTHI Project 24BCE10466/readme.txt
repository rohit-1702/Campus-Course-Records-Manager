Campus Course Records Manager - Java Project

How to Run:
1. Import the 'CampusRecordManager' folder as a Java project in Eclipse IDE.
2. Ensure Java JDK 11 or above is installed on your system.
3. Build/compile the project inside Eclipse.
4. Run the main class: edu.ccrm.cli.Main
5. Use the menu for managing student and course records.

Project Overview:
- This project manages student and course records including CRUD operations.
- The interface is console-based and menu-driven for easy interaction.
- Core classes are organized into domain, service, and CLI packages.

Requirements:
- Java Development Kit (JDK) 11 or higher
- Eclipse IDE or compatible Java IDE
